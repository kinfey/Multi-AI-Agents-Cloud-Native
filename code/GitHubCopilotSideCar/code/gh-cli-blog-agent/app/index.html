📝 Blog Agent - Blog Viewer

✅ App Running

Architecture: Kubernetes Dual-Sidecar Pattern

  • Main Container (blog-app): Nginx server displaying generated blog content on port 80
  • Sidecar 1 (copilot-agent): FastAPI + GitHub Copilot SDK, generates blog content on port 8001
  • Sidecar 2 (skill-server): FastAPI skill management, serves SKILL.md on port 8002
  • Shared Volumes: blog-data (sidecar→app) + skills-shared (skill→sidecar)

Data Flow: Skill Server syncs skills → Copilot Agent reads skills & generates blogs → writes to shared volume → App serves content

📂 Generated Blogs

Blog files generated by the Copilot Agent sidecar are available at /blog/

🔗 Endpoints

📄 Browse Blog Files (Main App) 🤖 Copilot Agent API (Sidecar 1) ❤️ Agent Health Check 📋 Blog List API 📚 Skill Server Health (Sidecar 2) 📑 Skill List API
